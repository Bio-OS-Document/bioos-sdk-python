submit(data_model_name: str, row_ids: List[str], inputs: str, outputs: str, submission_desc: str, call_caching: bool, submission_name_suffix: str = '') → List[Run]¶

Submit an existed workflow.

Example:

ws = bioos.workspace("foo")
wf = ws.workflow(name="123456788")
wf.submit(inputs = "{"aaa":"bbb"}",
          outputs = "{"ccc": "ddd"}",
          data_model_name = "bar",
          row_ids = ["1a","2b"],
          submission_desc = "baz",
          call_caching = True)
Parameters:

  • data_model_name (str) – The name of data_model to be used

  • row_ids (List[str]) – Rows to be used of specified data_model

  • inputs (str) – Workflow inputs

  • outputs (str) – Workflow outputs

  • submission_name_suffix (str) – The suffix of this submission’s name, defaults to yyyy-mm-dd-HH-MM-ss. The name format is {workflow_name}-history-{submission_name_suffix}

  • submission_desc (str) – The description of this submission

  • call_caching (bool) – CallCaching searches in the cache of previously running tasks with exactly the same commands and exactly the same input tasks. If the cache hit, the results of the previous task will be used instead of reorganizing, thereby saving time and resources.

Returns:

Result Runs corresponding to submitted workflows

Return type:

List[Run]
